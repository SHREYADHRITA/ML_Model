# ML_Model
This repository contains various models to analyse a linear classification model and predict the output using that model

Dataset used for creating this model- https://archive.ics.uci.edu/dataset/529/early+stage+diabetes+risk+prediction+dataset

Instuctions to run the code:

Run using the following command ---->
        streamlit run {FILE_PATH}
    For Example - 
        streamlit run D:\github_repo\ML_Model\streamlit_app.py
